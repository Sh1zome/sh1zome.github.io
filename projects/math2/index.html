Теорема Пифагора

Основные понятия

Теорема Пифагора — в прямоугольном треугольнике квадрат длины гипотенузы равен сумме квадратов длин катетов.

Гипотенуза — сторона, лежащая напротив прямого угла.

Катет — одна из двух сторон, образующих прямой угол.

Формула Теоремы Пифагора

\(a^2 + b^2 = c^2\)

a, b — катеты, с — гипотенуза.

Из этой формулы можно вывести следующее:

1. \(a = {\sqrt{с^2 - b^2}}\)

2. \(b = {\sqrt{с^2 - a^2}}\)

3. \(c = {\sqrt{a^2 - b^2}}\)

Для треугольника со сторонами a, b и c, где c — большая сторона, действуют следующие правила:

1. если \(c^2 < a^2 + b^2\), значит угол, противолежащий стороне c, является острым.

2. если \(c^2 = a^2 + b^2\), значит угол, противолежащий стороне c, является прямым.

3. если \(c^2 > a^2 + b^2\), значит угол, противолежащий стороне c, является тупым.
